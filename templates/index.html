📚 API Documentation

Complete REST API documentation for developers.

Base URL

http://localhost:8000/api/

Available Endpoints

Voting

GET /voting/procedures/ - List all voting procedures
GET /voting/methods/ - List all voting methods

Candidates

GET /candidates/ - List all candidates
POST /candidates/comparisons/ - Create a candidate comparison

Registration

GET /registration/requirements/ - List registration requirements
GET /registration/methods/ - List registration methods
GET /registration/deadlines/ - List registration deadlines

Polling

GET /polling/locations/ - List polling locations
GET /polling/early-voting/ - List early voting locations

Getting Started

To use this platform:

  1. Install dependencies: pip install -r requirements.txt
  2. Run migrations: python manage.py migrate
  3. Create a superuser: python manage.py createsuperuser
  4. Load sample data (optional)
  5. Start the server: python manage.py runserver
  6. Visit Admin panel to manage content
